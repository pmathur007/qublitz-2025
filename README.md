# qublitz-2025

Please choose **one** of the following challenges to submit to!
* Beginner: Complete a series of introductory challenges to get familiar with quantum computing and Qiskit!
* Hardware: Replicate the results of the 2025 Nobel Prize in Physics Experiment!
* Algorithms: Use the Jordan-Wigner Transform and VQE algorithm to find the ground state energy of molecules!

Good luck!
